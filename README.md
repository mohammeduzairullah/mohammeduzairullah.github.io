# 💼 Modern IT Project Management & Coordination Portfolio

A sleek, dark-themed, high-performance professional portfolio tailored for an aspiring **Project Coordinator** and future **IT Project Manager**. Built with a focus on clean systems-thinking, modern web design aesthetics, and structured tracking.

---

## 🚀 Key Features

*   **Tailwind CSS Structure:** Fully responsive, fluid mobile-to-desktop grid system designed around a sleek dark slate theme.
*   **Agile & PM Focused Copywriting:** Tailored explicitly to showcase knowledge in Agile frameworks, the Scrum framework, Jira tracking, risk management, and team collaboration.
*   **Interactive Mobile Links:** Fully integrated `mailto:` protocols and optimized `tel:` actions for instant mobile-dialer execution.
*   **Integrated Google Forms Channel:** Connected to a dedicated background-free response workflow to log inbound professional inquiries instantly.
*   **Inspect Element Deterrent:** Embedded JavaScript to block basic browser inspection shortcuts (`F12`, `Ctrl+Shift+I`, right-click) to protect layout integrity.

---

## 🛠️ Built With

*   **Frontend HTML5:** Clean, semantic structure mapped with distinct navigational regions (`#about`, `#skills`, `#projects`, `#certifications`).
*   **Tailwind CSS (via CDN):** Rapid structural assembly using responsive modifiers (`md:grid-cols-2`, `backdrop-blur-md`).
*   **Google Fonts API:** `Plus Jakarta Sans` for geometric, readable prose paired with `Playfair Display` for editorial accents.
*   **JavaScript (ES6):** Custom event listeners capturing keyboard patterns and pointer inputs for contextual security.
